# Kreya Project Guidelines

## Project Overview
- Purpose: AI-powered WhatsApp-first social media management SaaS.
- Tech Stack: Static HTML/CSS/JS (GitHub Pages).

## Development Rules
- **Conciseness:** Do NOT rewrite whole files. Only provide the specific code blocks that need changing.
- **Artifacts:** When creating new artifacts, use the existing brand styles defined in `kreya-brand-identity.html`.
- **Memory Management:** If a chat session gets long, summarize our progress and ask me to start a new chat.
